# Теоретическое введение

Markdown — это язык разметки с облегчённым синтаксисом, предназначенный для простого форматирования текста. Он позволяет быстро создавать структурированные документы, которые могут быть преобразованы в HTML, PDF, DOCX и другие форматы.

Основные элементы синтаксиса Markdown:
- Заголовки: `# Заголовок 1`, `## Заголовок 2`
- Выделение текста: **жирный** (`**текст**`), *курсив* (`*текст*`)
- Списки: нумерованные (`1. пункт`) и маркированные (`- пункт`)
- Ссылки: `[текст](адрес)`
- Изображения: `![подпись](путь/к/изображению)`
- Формулы LaTeX: внутристрочные `$формула$`, выносные `$$формула$$`
- Цитаты: `> текст цитаты`
- Блоки кода: обратные апострофы или тройные апострофы с указанием языка

Для компиляции отчётов используется система Quarto, которая поддерживает расширенные возможности: перекрёстные ссылки, библиографию, шаблоны оформления.
